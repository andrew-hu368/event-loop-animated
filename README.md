### Prerequisites

* `node 18.12.0` 
* `npm >= 6.14.15` 


### Configuration

1. Install dependencies
   ```sh
   npm install
   ```
2. Start the PostgreSQL database
   ```sh
   docker-compose up -d
   ```
3. Seed the database
   ```sh
   npm run seed
   ```
